⚙️ Client Konfigurationsbeispiele

Trage diese Server-Konfiguration in deinen KI-Assistenten ein. Die Beispiele zeigen den aktuellen Server-Host und die generierten Schlüssel.

1. Antigravity CLI (agy) & Antigravity IDE

Füge dies unter mcpServers in deiner ~/.gemini/config/mcp_config.json ein:

{
  "mcpServers": {
    "llmwiking-okf": {
      "type": "sse",
      "url": "{{ request.base_url }}LLMWikiNG/mcp/sse",
      "env": {
        "X-MCP-Key": "{{ new_generated_mcp_key or mcp_config.get('llmwiking_mcp_key') or 'DEIN_MCP_KEY' }}",
        "X-API-Key": "{{ new_generated_api_key or 'DEIN_API_KEY' }}"
      }
    }
  }
}
2. OpenCode

Füge dies unter mcp in deiner opencode.json Konfiguration hinzu:

{
  "mcp": {
    "llmwiking-okf": {
      "type": "remote",
      "url": "{{ request.base_url }}LLMWikiNG/mcp/sse",
      "enabled": true,
      "environment": {
        "X-MCP-Key": "{{ new_generated_mcp_key or mcp_config.get('llmwiking_mcp_key') or 'DEIN_MCP_KEY' }}",
        "X-API-Key": "{{ new_generated_api_key or 'DEIN_API_KEY' }}"
      }
    }
  }
}
