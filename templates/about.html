{{ _('about.heading') }}

LLMWikiNG (OKF Edition) is a local, privacy-friendly wiki platform developed by ZeroDot1. It is based on the standardized Open Knowledge Format (OKF) v0.1 and inspired by Andrej Karpathy's LLM-Wiki-Pattern. The wiki is built, maintained, and extended by intelligent LLM agents – and you can easily view, search, and manage it directly from your web browser.

💛 Support the project: I put a lot of time, money, and passion into developing this project to provide a free and privacy-focused knowledge base. Every little bit of support helps me immensely to keep improving LLMWikiNG! If you would like to help, feel free to check out my Amazon Wishlist. Thank you so much!

Special thanks to https://github.com/tevsa for the great idea and awesome support in building this project!

{{ _('about.features_heading') }}

  • 📦 Open Knowledge Format (OKF) – Fully compliant with the OKF v0.1 specification (YAML frontmatter, concepts, resources, and structured paths).
  • 🔗 Native Markdown Links – Uses standardized Markdown links ([Label](/path/to/page.md)) instead of proprietary Wikilinks.
  • 🗂️ Multi-Wiki Support – Create and switch between completely separate wikis stored under wikis/<Name>/, managed via the WebUI.
  • ✍️ Universal Editor (WYSIWYG & Markdown) – Integrated dual-mode editor to create and edit wiki pages and raw sources directly in the browser.
  • 🕸️ Knowledge Graph – Interactive, color-coded visual network map of all page relations (fully offline – custom dependency-free Canvas engine, no external libraries). Optimized for large wikis with Lazy Loading, Barnes-Hut O(n log n) physics, Viewport Culling, and Level-of-Detail rendering. The graph page includes a node search, tag filter bar, node detail panel, stats overlay, zoom buttons, fullscreen mode, and a wiki switcher in the toolbar. Layout optimized and styled in a standalone stylesheet (graph.css).
  • 📜 Activity Timeline – Chronological list of recent ingests, deletions, and exports displayed right on the dashboard.
  • 📰 Weekly Reports & Email Briefings – Aggregates weekly wiki changes and sends them securely via an integrated SMTP client.
  • 🔍 Full-text Search – Blazing fast BM25 hybrid search inside the wiki, raw sources, and exported documents with term highlighting.
  • ⏳ Pending Ingest – Import manually placed files into the wiki individually or in batches with a single click.
  • 📤 Export Management – Copy wiki pages to the export folder to view or download them.
  • 📥 Network & Web Ingest – Support for manual file uploads, text pastes, URL notes, as well as a fully automated direct remote network ingest via curl or client scripts using secure API keys.
  • 🩹 Health Check (Lint) – Detects orphan pages, missing link targets, and stale pages.
  • 🌗 Themes (Appearance) – Dark or light theme, stored persistently in config.json (Dark Mode is default).
  • 🔐 Authentication & API – Secure login with Argon2 hashes, automatic setup redirection, auto-generated API keys on registration, full API-key management with secure password-based **API-Key Recovery**, and a robust JSON API under /api/v1.
  • ⬇️ Self-Update – Integrated update utility to pull the latest version from GitHub with one click. Safely performs backups of databases and configurations to /tmp and restores them automatically after git reset.
  • 🐋 Docker & UGreen NAS Optimization – Docker setup specifically optimized for UGreen NAS (UGOS) to support automatic volume detection and out-of-the-box compatibility. Can also be deployed locally using the DOCKER_VOLUME_BASE path variable.
  • ⚙️ Settings – Central configuration panel with tabs for language, theme, SMTP, user management, API keys, registration controls, health, and updates.
